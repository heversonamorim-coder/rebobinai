# Imagens dos produtos físicos

Coloque aqui as fotos dos produtos que aparecem no checkout (plano
"+Lembrança física"). Os nomes precisam bater com o catálogo
(`apps/web/src/lib/products.ts`):

- `caneca.webp` — foto da caneca personalizada
- `camiseta.webp` — foto da camiseta

Sugestão: quadradas (ex.: 800×800), fundo neutro, formato **.webp** (leve).
Enquanto o arquivo não existir, o checkout mostra um placeholder com o emoji
do produto — nada quebra.
